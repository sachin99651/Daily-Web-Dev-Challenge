# 🏗️ Module 05: JavaScript Objects

## 📌 Overview
Objects are one of the most important parts of JavaScript. In this module, I learned how to group related data and functions together into a single entity. This is the foundation for handling "Models" in the MERN stack (like User, Product, or Order objects).

## 📂 Featured Labs
* **`objectMethods.js`**: Practicing how to add functions (methods) inside an object.
* **`nestedObjects.js`**: Managing complex data structures where objects contain other objects or arrays.
* **`propertyAccess.js`**: Using Dot Notation and Bracket Notation to retrieve and update data.

## 🛠️ Concepts Mastered
- **Key-Value Pairs:** Understanding the structure of an object.
- **Dot vs. Bracket Notation:** Knowing when to use `object.name` vs `object["name"]`.
- **`this` Keyword:** Using the `this` keyword to access properties from within an object's method.
- **Object Manipulation:** Adding, deleting, and updating properties dynamically.
