# 🎨 CSS Practice Labs

## 📌 Overview
This folder contains practical implementations of CSS concepts learned during the Responsive Web Design certification. Each sub-folder represents a specific UI component or layout challenge.

## 📂 Lab Breakdown
* **`colorBox`**: Practicing the CSS Box Model, margins, padding, and border properties.
* **`colors`**: Exploring color theory, hex codes, RGB, and HSL values in CSS.
* **`form`**: Styling interactive HTML elements and practicing input validation UI.
* **`gameSetting`**: Building a UI settings panel to practice layout positioning and flexbox.

## 🛠️ Concepts Applied
- **Selectors:** Using classes, IDs, and element selectors.
- **Box Model:** Mastering the spatial relationship between elements.
- **Visual Design:** Implementing custom color palettes and typography.
